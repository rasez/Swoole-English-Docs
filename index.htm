Swoole Extension Manual v1.8.10

PHP's asynchronous, parallel, high-performance network communication engine, written in pure C language, provides asynchronous multi-threaded server in PHP language, asynchronous TCP/UDP network client, asynchronous MySQL, asynchronous Redis, database connection pool, AsyncTask, message queue , millisecond timer, asynchronous file read and write, asynchronous DNS query. Swoole has built-in Http/WebSocket server/client and Http2.0 server.

Swoole can be widely used in the Internet, mobile communications, enterprise software, online games, Internet of Things, car networking, smart home and other fields. Using PHP+Swoole as a network communication framework can greatly enhance the efficiency of enterprise IT R&D teams and focus more on developing innovative products.

The underlying Swoole has an asynchronous non-blocking, multi-threaded network IO server built in. PHP programmers only need to handle event callbacks, no need to care about the underlying. Unlike the fully asynchronous frameworks such as Nginx/Tornado/Node.js, Swoole supports both full asynchronous and synchronous.

Swoole is open source free free software, the license agreement is Apache2.0. Swoole code is freely available to businesses and developers, and modifications made on Swoole are available for commercial products without the need for open source.
