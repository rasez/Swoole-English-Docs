swoole_timer_add

Add a timer for the fpm/apache/cli environment.

swoole_timer_add($interval, $callback);
 
swoole_timer_add(3000, function($interval) {
    echo "timer[$interval] :".date("H:i:s")." call\n";
});
  • $interval is the timer interval, the unit is milliseconds, there can be 2 timers with the same time interval.
  • $callback is the event callback function of the timer
  • Need swoole-1.6.12+
  • Not available in swoole_server
  • Remove this timer from version 1.8.0 or higher, please use swoole_timer_tick
  • Timers must be used in full-asynchronous mode. Do not use in synchronous blocking code.
