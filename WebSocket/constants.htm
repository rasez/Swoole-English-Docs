Predefined constant

WebSocket data frame type

  • WEBSOCKET_OPCODE_TEXT_FRAME = 0x1, UTF-8 text character data
  • WEBSOCKET_OPCODE_BINARY_FRAME = 0x2, binary data

WebSocket connection status

  • WEBSOCKET_STATUS_CONNECTION = 1, the connection enters waiting for a handshake
  • WEBSOCKET_STATUS_HANDSHAKE = 2, shaking hands
  • WEBSOCKET_STATUS_FRAME = 3, the handshake has been successful, waiting for the browser to send data frames
