How to determine if the connection is a WebSocket client

By using $server->connection_info to get the connection information, one of the returned arrays is websocket_status. According to this state, it can be judged whether it is a WebSocket client.

  • WEBSOCKET_STATUS_CONNECTION = 1, the connection enters waiting for a handshake
  • WEBSOCKET_STATUS_HANDSHAKE = 2, shaking hands
  • WEBSOCKET_STATUS_FRAME = 3, the handshake has been successful, waiting for the browser to send data frames
