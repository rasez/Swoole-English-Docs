swoole_websocket_server->push

Push data to the websocket client connection, the maximum length must not exceed 2M.

function swoole_websocket_server->push(int $fd, string $data, int $opcode = 1, bool $finish = true);
  • $fd ID of the client connection. If the specified TCP connection corresponding to $fd is not a websocket client, it will fail to send.
  • $data data content to send
  • $opcode, which specifies the format in which the data content is sent. The default is text. Send binary content $opcode parameter needs to be set toWEBSOCKET_OPCODE_BINARY_FRAME
  • True if the send succeeds, false if the send fails

Swoole_websocket_server->push is available in swoole-1.7.11 or later
