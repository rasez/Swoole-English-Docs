onOpen

This function is called back when the WebSocket client establishes a connection with the server and completes the handshake.

function onOpen(swoole_websocket_server $svr, swoole_http_request $req);
  • $req is an Http request object containing the handshake request information sent by the client.
  • The onOpen event function can call push to send data to the client or close to close the connection.
  • onOpen event callback is optional
