Optimize swoole memory allocation performance with jemalloc

About jemalloc

Jemalloc is a more efficient memory pool technology than glibc malloc, which is used extensively on Facebook and uses jemalloc as the default memory manager in FreeBSD and FireFox projects.

Unlike tcmalloc, jemalloc is developed using pure C language. And tcmalloc is developed in C++.

Install jemalloc

  • GITHUB homepage: https://github.com/jemalloc/jemalloc
  • Download address: https://github.com/jemalloc/jemalloc/releases/tag/4.0.4

Compile and install:

./configure
    sudo make -j install
    sudo ldconfig

Using jemalloc

The 1.7.x version manually changes the Makefile to the swoole-srcdirectory modification Makefilefile, SWOOLE_SHARED_LIBADDappends it at the end -ljemalloc, and then recompiles and installs the swoole to jemallocoptimize the performance of memory allocation.

1.8.0 or higher compilation parameter increase--enable-jemalloc
