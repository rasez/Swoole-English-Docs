Those functions in PHP are synchronously blocked.

Synchronous blocking function

  • Mysql, mysqli, pdo and other DB operation functions
  • sleep、usleep
  • curl
  • Stream, socket extension function
  • Swoole_client synchronization mode
  • Memcache, redis extension function
  • File_get_contents/fread and other file read functions
  • swoole_server->taskwait
  • swoole_server->sendwait

The swoole_server PHP code has the above function, Server is the synchronization server
. The above function is asynchronous server.

Asynchronous non-blocking function

  • Swoole_client asynchronous mode
  • Mysql-async library
  • Redis-async library
  • swoole_timer_tick/swoole_timer_after
  • Swoole_event series function
  • swoole_table/swoole_atomic/swoole_buffer
  • swoole_server->task/finish函数
