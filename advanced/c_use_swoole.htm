How C/C++ developers use Swoole

Swoole uses cmake to compile the configuration. The sample program is in examples/server.c. You can develop your code on this basis. If you need to modify the compilation details, please modify CMakeLists.txt directly.

Build & Install

cmake .
make
make install

Example

Sample code: examples/server.c In the C/C++ code, you only need to introduce the swoole header.

#include <swoole/Server.h>
#include <swoole/Client.h>
 
int main()
{
    swServer serv;
    swServer_create(&serv);
    serv.onStart = my_onStart;   
    ...
    swServer_start(&serv);
}

Compile and run

gcc -o server server.c -lswoole
./server
