Predefined constant

  • SWOOLE_VERSION当前Swoole的版本号，字符串类型，如1.6.0
  • new swoole_server 构造函数参数
    • SWOOLE_BASE uses the Base mode, the business code is executed directly in Reactor
    • SWOOLE_PROCESS uses process mode, business code is executed in the worker process
  • new swoole_client 构造函数参数
    • SWOOLE_SOCK_TCP creates tcp socket
    • SWOOLE_SOCK_TCP6 creates tcp ipv6 socket
    • SWOOLE_SOCK_UDP creates udp socket
    • SWOOLE_SOCK_UDP6 creates udp ipv6 socket
    • SWOOLE_SOCK_SYNC synchronization client
    • SWOOLE_SOCK_ASYNC asynchronous client
  • new swoole_lock构造函数参数
    • SWOOLE_FILELOCK creates a file lock
    • SWOOLE_MUTEX creates a mutex
    • SWOOLE_RWLOCK creates a read-write lock
    • SWOOLE_SPINLOCK creates a spin lock
    • SWOOLE_SEM creates semaphores
  • SSL加密方法
    • SWOOLE_SSLv3_METHOD
    • SWOOLE_SSLv3_SERVER_METHOD
    • SWOOLE_SSLv3_CLIENT_METHOD
    • SWOOLE_SSLv23_METHOD (default encryption method)
    • SWOOLE_SSLv23_SERVER_METHOD
    • SWOOLE_SSLv23_CLIENT_METHOD
    • SWOOLE_TLSv1_METHOD
    • SWOOLE_TLSv1_SERVER_METHOD
    • SWOOLE_TLSv1_CLIENT_METHOD
    • SWOOLE_TLSv1_1_METHOD
    • SWOOLE_TLSv1_1_SERVER_METHOD
    • SWOOLE_TLSv1_1_CLIENT_METHOD

    • SWOOLE_TLSv1_2_SERVER_METHOD
    • SWOOLE_TLSv1_2_CLIENT_METHOD

    • SWOOLE_DTLSv1_SERVER_METHOD
    • SWOOLE_DTLSv1_CLIENT_METHOD
