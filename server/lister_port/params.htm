Optional parameter

The listening port calling setmethod can only set some specific parameters, and cannot modify the global Server settings.

  • The listening port does not set any parameters, it will inherit the relevant configuration of the primary server.
  • Http/WebSocket server, if the protocol parameters are not set, the listening port will still be set to Http or WebSocket protocol, and the onReceivecallback set for the port will not be executed.

List of available parameters

  • socket参数，如backlog、TCP_KEEPALIVE、open_tcp_nodelay、tcp_defer_accept等
  • Protocol related, such as open_length_check, open_eof_check, package_length_type, etc.
  • SSL certificate related, such as ssl_cert_file, ssl_key_file, etc.

List of parameters that are not available

  • worker_num、task_worker_num、reactor_num
  • dispatch_mode、task_ipc_num
  • heartbeart_check
  • log_file
  • user/group/chroot
  • open_cpu_affinity
  • max_request/task_max_request
