swoole_http_request->$files

文件上传信息。类型为以form名称为key的二维数组。与PHP的$_FILES相同。 

Array
(
    [name] => facepalm.jpg
    [type] => image/jpeg
    [tmp_name] => /tmp/phpn3FmFr
    [error] => 0
    [size] => 15476
)
  • name 浏览器上传时传入的文件名称
  • type MIME类型
  • tmp_name 上传的临时文件，文件名以/tmp/swoole.upfile开头
  • size 文件尺寸

在1.7.16以上版本可用

上传的临时文件在调用$response->end后，会自动删除
